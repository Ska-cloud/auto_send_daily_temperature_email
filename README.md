## 使用说明
编写config.xml文件：配置发件人信息、收件人信息、每天发邮件的时间，需要自行申请qq邮箱的pop3码。配制好后直接运行<br />
`python run.py`
## docker部署
打包镜像：`docker build -t xxx .`<br />
启动项目：`docker run --name xxx -itd image_id`
## 配置环境
`pip install -r requirements.txt`
